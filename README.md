# NID Voter Backend

Express backend for the NID voter application.

## Setup

```bash
npm install
```

## Run

```bash
npm start
```

## cPanel Startup File

Use `app.js` as the application startup file in cPanel Node.js App setup.

## Environment

- Copy `.env.example` to `.env` if needed
- Set your MongoDB connection in `.env`
- Change `AUTH_USERNAME` and password hash for production
